What I'm Working On

My current focus is the Control Plane: a private infrastructure for orchestrating AI coding agents with full audit trails. It includes a file-system activity tracker with CRDT-based versioning (Go + SQLite), a multi-agent task orchestrator that routes work to Claude Code, Gemini CLI, and Codex with progressive autonomy controls, and a declarative configuration engine that generates tool-specific settings from org-mode files.

Side projects: an NHL Data Graph tracking trade lineage and player analytics, a Spotify playlist manager that auto-removes played songs, and a local voice tools system running faster-whisper on a RTX 3090 for on-device speech-to-text.

The underlying conviction is simple: state should be derived from events, text files are the universal interface for agent coordination, and autonomy without audit trails is dangerous.
