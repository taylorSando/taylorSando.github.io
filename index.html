What I'm Working On

My current focus is the Control Plane: a private infrastructure for orchestrating AI coding agents with full audit trails. It includes a file-system activity tracker with CRDT-based versioning, a multi-agent task orchestrator with progressive autonomy controls, and a declarative configuration engine that generates tool-specific settings from org-mode files.

The underlying conviction is simple: state should be derived from events, text files are the universal interface for agent coordination, and autonomy without audit trails is dangerous.
