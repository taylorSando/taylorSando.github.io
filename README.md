# taylorsando.com

Personal site and consulting presence. Jekyll on GitHub Pages.

**Live:** [taylorsando.com](https://www.taylorsando.com)

## What's here

- Landing page with consulting positioning, services, and experience timeline
- Writing section with essays on agent orchestration, testing epistemology, and distributed systems
- About page tracing the arc from psychology research to AI agent infrastructure

## Stack

- Jekyll (GitHub Pages)
- Pure CSS, no JavaScript
- System font stack, dark mode via `prefers-color-scheme`
- Single CSS file (~450 lines)

## Local development

```
bundle exec jekyll serve
```

Or just push to `master` -- GitHub Pages builds automatically.
